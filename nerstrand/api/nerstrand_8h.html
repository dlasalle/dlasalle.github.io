Generate a clustering of a graph with an unspecified number of clusters.

Parameters
r_nvtxsA pointer to the number of vertices in the graph.
xadjThe start of the adjacency list of each vertex.
adjncyThe vertex at the far end of each edge, indexed by xadj.
adjwgtThe weight of each edge, indexed by xadj.
r_nclustersA pointer to the number of clusters.
cidThe cluster assignment for each vertex.
r_modA pointer to the modularity of the generated clustering.
Returns
NERSTRAND_SUCCESS unless an error is encountered.
